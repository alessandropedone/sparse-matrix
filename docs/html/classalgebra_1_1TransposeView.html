Detailed Description

template<AddMulType T, StorageOrder S = StorageOrder::RowMajor>
class algebra::TransposeView< T, S >

A view that represents the transpose of a given matrix.

The TransposeView class provides a transposed view of a matrix without copying its data. All operations are redirected to the underlying matrix with row and column indices swapped. This class supports both general and square matrices, and transparently handles compressed formats.

Template Parameters
TThe type of the matrix elements.
SThe storage type or additional matrix traits.
Note
The TransposeView does not own the underlying matrix unless constructed with dimensions, in which case it creates a new matrix.
