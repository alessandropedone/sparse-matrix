Detailed Description

Defines storage formats and utilities for sparse matrices.

Author

This header provides various storage structures and utilities for representing sparse matrices in different formats, including compressed, modified compressed, and uncompressed (map-based) storage. It also defines concepts and type traits for handling numeric and complex types, as well as storage order (row-major or column-major).

The main components of this file are:

  • algebra::StorageOrder : Enum for specifying matrix storage order.
  • algebra::is_complex : Type trait to detect std::complex types.
  • algebra::AbsReturnType : Type trait to determine the return type of std::abs.
  • algebra::AddMulType : Concept for types supporting addition, multiplication, and absolute value.
  • algebra::CompressedStorage : Structure for compressed sparse matrix storage (CSR/CSC).
  • algebra::ModifiedCompressedStorage : Structure for modified compressed storage with explicit diagonal.
  • algebra::Index : Struct representing a matrix index (row, col).
  • algebra::RowMajor, algebra::ColMajor : Comparators for index ordering.
  • algebra::ComparatorSelector : Selector for comparator based on storage order.
  • algebra::UncompressedStorage : Alias for map-based sparse matrix storage.
