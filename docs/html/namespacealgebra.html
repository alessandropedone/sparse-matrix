Detailed Description

Contains classes and functions for algebraic operations on matrices, including various matrix representations, views, and operations such as multiplication, transposition, and norm calculation.

The algebra namespace provides a flexible and extensible framework for working with sparse and dense matrices in different storage formats (row-major, column-major), supporting efficient mathematical operations and matrix manipulations.

Enumeration Type Documentation

◆ NormType

enum class algebra::NormType
strong

Enumeration of supported matrix norm types.

This enum defines the types of norms that can be calculated for a matrix:

  • One: The maximum absolute column sum of the matrix.
  • Infinity: The maximum absolute row sum of the matrix.
  • Frobenius: The square root of the sum of the absolute squares of its elements.

◆ StorageOrder

enum class algebra::StorageOrder
strong

Enum class to specify the storage order of a matrix.

This enum is used to indicate whether the matrix is stored in row-major or column-major order.

  • RowMajor: Matrix elements are stored row by row.
  • ColumnMajor: Matrix elements are stored column by column.
