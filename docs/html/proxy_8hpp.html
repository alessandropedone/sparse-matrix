Detailed Description

Defines the Proxy class for sparse matrix element access and modification.

Author

This file contains the declaration and implementation of the algebra::Proxy class template, which provides a proxy object for accessing and modifying elements in a sparse matrix. The Proxy enforces sparse storage rules by ensuring that zero values are not explicitly stored in the underlying uncompressed storage format. Assignments and arithmetic operations are intercepted to insert, update, or erase elements as appropriate.
