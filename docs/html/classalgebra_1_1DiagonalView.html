Detailed Description

template<AddMulType T, StorageOrder S = StorageOrder::RowMajor>
class algebra::DiagonalView< T, S >

A view that represents the diagonal of a square matrix.

The DiagonalView class provides a view of the diagonal elements of a square matrix, treating all off-diagonal elements as zero. This class supports both general and square matrices, and transparently handles compressed formats.

Template Parameters
TThe type of the matrix elements.
SThe storage type or additional matrix traits.
Note
The DiagonalView does not own the underlying matrix unless constructed with dimensions, in which case it creates a new matrix.
