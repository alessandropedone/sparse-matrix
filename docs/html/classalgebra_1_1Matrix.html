Detailed Description

template<AddMulType T, StorageOrder S = StorageOrder::RowMajor>
class algebra::Matrix< T, S >

Represents a sparse matrix with configurable storage order and element type.

The Matrix class provides an interface for constructing, manipulating, and performing operations on sparse matrices. It supports both row-major and column-major storage orders, and can store elements of any type that supports addition and multiplication. The class allows for dynamic construction, compression (to CSR/CSC), and uncompression (to COO), as well as efficient multiplication with vectors and other matrices. It also provides views for transposed and diagonal matrices, and supports parallel compression.

Template Parameters
TType of the matrix elements. Must satisfy AddMulType concept.
SStorage order of the matrix (RowMajor or ColumnMajor). Defaults to RowMajor.
Note
The Matrix class inherits from AbstractMatrix and is not default-constructible.
